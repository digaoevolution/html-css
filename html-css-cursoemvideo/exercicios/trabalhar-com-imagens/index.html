Adicionar imagens, favicon, musicas

para adicionar imagens usa a tag img - mas primeiro temos que adicionar os arquivos de imagem a serem usados na pasta de uso atual

para adicionar o faveicom (o icone que aparece na aba do navegador) temos que usar a tag link:favicon

assim que adicionar a tag desejada com o cursor piscando preciona Ctrl+space e seleciona a imagem
